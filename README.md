## Installation

```bash
npm install @spjpgrd/inkwell
# or
yarn add @spjpgrd/inkwell
# or
pnpm add @spjpgrd/inkwell
```

## Usage

```typescript
import { amber, blue, gray } from '@spjpgrd/inkwell';

// Use the colors in your CSS
const styles = {
  backgroundColor: amber.amber9,
  color: gray.gray1,
  borderColor: blue.blue6,
};
```
